This project analyzes Amazon delivery performance using **driver** and **eMentor** metrics.  
It performs data cleaning, metric calculations, and visualizations to extract insights such as:

- Average bags / parcels / oversize per day
- Mileage discrepancies between driver and eMentor
- Delivery score trends
- Stops vs performance correlation
- Extra mileage distribution
